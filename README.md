This content is a self-paced workshop introducing the components of the `k0s family`:  

- [k0s](https://k0sproject.io) a lightweight Kubernetes distribution
- [k0sctl](https://github.com/k0sproject/k0sctl) allows you to manage cluster lifecycles
- [k0smotron](https://k0smotron.io/) enables Kubernetes control planes to run inside Pods
- [k0rdent](https://kordent.io) provides the building blocks for Kubernetes-based internal developer platforms

You can browse the Hugo-based content at [https://k0s-family.luc.run](https://k0s-family.luc.run)

:warning: This is still a work in progress, feedback more than welcome. Please [create an issue](https://github.com/lucj/k0s-family/issues/new) if you notice something wrong.
